# AI-Powered Facial Analysis ✨📸

A futuristic, high-fidelity UI/UX mobile application prototype designed for AI-driven skincare and biometric facial analysis. 

Built with a modern **cyber-luminescent** dark aesthetic, this concept features multiple app screens seamlessly structured into
a single responsive container, showcasing how an advanced AI diagnostic interface looks and feels.

---

## 📱 Interactive Screens Included

*   **Screen 1: Onboarding Landing:** An immersive welcome view featuring an active neon portrait scanning ring and minimalist brand layout.
*   **Screen 2: Consultation / Skin Type:** A sleek card-based preference selection layout with custom neon radio check indicators.
*   **Screen 3: Biometric Scan Engine:** A simulated face-scanning HUD viewport with dynamic horizontal scan lines, bounding boxes, and lighting validation indicators.
*   **Screen 4: Analytics Report:** A clean data-visualization screen featuring analysis gauges, barrier health metrics, and comparison tiles.
*   **Screen 5: Personalized Lab:** A custom recommendation view with category filter chips, product display cards, and an integrated bottom navigation bar.

---

## 🛠️ Tech Stack & Architecture

*   **Markup:** Clean, semantic HTML5 structure.
*   **Styling:** Modern CSS3 leveraging global CSS Custom Properties (`:root`) for absolute control over neon color tokens and shadows.
*   **Typography:** Google Fonts integration (`Cormorant Garamond`, `DM Sans`, and `Space Mono`).
*   **Effects & Animations:** Pure CSS keyframes for biometric scanner loops, custom SVG noise layers for texture, and advanced `backdrop-filter` for glassmorphism.
